주제 : 게시글 요구사항 명세서

- 메인메뉴
	1. 관리자 메뉴
	2. 회원 메뉴
	

- 회원가입
	- 게시판은 회원만이 이용할 수 있다
	- 관리자는 회원가입 필요가 없으며 이용자만 회원가입을 한다
	- 회원가입 시 필요한 요구 사항
		- 아이디 : 아이디는 중복이 불가능하다 (관리자)
		- 비밀번호	: 비밀번호는 영문, 숫자, 특수문자(!@#$%) 만 사용 가능하며 
				12자 이하로 한다 (관리자)
		- 전화번호 : 전화번호는 010-1234-5678 형식을 취한다
		- 이메일 주소 : 이메일 주소는 asd123@naver.com 의 형식으로 영문과 숫자만 
				사용가능하다
- 로그인
	- 로그인 시 아이디와 비밀번호를 이용하여 로그인 한다
	- 로그인은 저장된 아이디와 비밀번호만 사용 가능하다
	- 같은 아이디가 로그인 되어 있으면 중복 사용은 불가능 하다
- 로그아웃
	- 사용이 끝난 사용자가 로그아웃을 선택하여 프로그램을 종료한다
	- 프로그램 종료는 번호를 선택하여 종료한다
- 관리자 
	- 관리자는 회원가입한 회원을 저장하고 삭제할 수 있다
	- 카테고리
		- 카테고리는 관리자가 추가, 수정, 삭제할 수 있다
	- 게시판
		- 관리자는 게시판의 조회를 하여 이용자가 작성한 게시판을 삭제할 수 있다
- 이용자
	- 카테고리
		- 이용자는 로그인 후 게시판에 게시글을 작성할 수 있다
		- 자신이 작성한 게시글은 아이디를 검색하여 조회 가능하다
		- 자신이 작성한 게시글을 수정 및 삭제가 가능하다
- 조회수
	- 게시판에 등록된 게시글을 누를 때마다 1씩 조회수가 추가된다
	- 조회수는 수정이 불가능하고 초기화/ 조회만 가능하다
- 분류관리
	- 분류등록
		- 카테고리를 등록하여 카테고리 안에 제품을 등록할 수 있다
		- 카테고리는 중복될 수 없으며 관리자는 수정 및 삭제를 할 수 있다
		- 이용자는 카테고리를 선택만 가능하며 카테고리에 따른 글을 작성할 수 있다
	- 분류삭제
		- 관리자는 카테고리를 삭제 가능하며 삭제 시 분류에 따른 게시글 또한 삭제한다
