/* 전자기기 정보 사이트 
- 사용자(회원 + 비회원).user
	- 사용자는 사이트의 리뷰와 뉴스, 전문지식, 커뮤니티의 조회만 가능하다
		- 댓글이나 게시글 작성을 시도하려 할 때 "권한이 없습니다." 문구를 출력한다
        - 비회원은 댓글의 확인만 가능하다 
	- 유튜브 링크를 선택하면 해당 유튜브 채널로 이동한다 
- 회원.member
	- 게시글의 댓글을 작성할 수 있다
    - 회원은 리뷰 게시판, 뉴스 게시판, 전문지식 게시판, 커뮤니티 게시판의 게시글 조회 및 등록이 가능하다
    - 해당 회원이 작성한 게시글은 해당 회원만 수정, 삭제할 수 있다 
    - 회원이 공동구매 게시판을 이용했을 때 조회 및 참가가 가능하다 
		- 공동구매 게시글이 등록되었을 때 회원들에게 알림이 간다 
	- 사이트의 이벤트는 회원만 참여가능하며 이벤트를 관리자가 등록했을 때 회원들에게 알림이 간다 
- 관리자.admin
	- 관리자 회원 관리 
		- 관리자는 회원의 가입정보를 저장하여 필요 시 삭제가 가능하다  
	- 관리자는 게시판의 분류 생성/수정/삭제 가 가능하다 
    - 관리자만 숍과 이벤트 게시판에 게시글 작성/수정/삭제 가 가능하다 
    - 관리자는 광고 배너를 생성할 수 있다 
    - 관리자는 가입된 회원의 수를 조회 가능하다 
    - 관리자는 사이트의 하루 이용자 수를 조회 가능하다 
		- 하루 이용자 수는 게시글 조회 횟수에 따라 1씩 증가한다
	- 관리자는 게시글의 분류를 생성/ 수정/ 삭제 가 가능하다 
		분류는 스마트폰/ 테블릿/ 주변기기/ 컴퓨터/ 기타
- 회원가입.sign up
	- 회원가입 시 최초에 약관동의를 얻는다  
		- 약관동의는 회원의 삭제가 되었을 때의 경우를 입력한다   
			- 부적절한 게시글이나 댓글을 작성 했을 때 삭제될 수 있다 
            - 1년 동안 로그인이 안되었을 때 개인정보 보호를 위한 수신 동의를 회원에게 보내고 그 이후에도 로그인이 확인이 안되면 삭제가 가능하다 
   	 - 회원가입 시 기입항목 : 아이디/ 비번/ 이메일/ 이름/ 전화번호/ 주소 
   	 - 가입된 회원의 가입 정보는 관리자가 관리한다 
- 로그인.log in
	- 아이디와 비번을 입력하여 진행한다
	- 실패횟수는 5회 이며 횟수 소진 시 인증번호를 받은 후 아이디와 비번을 재수정 한다
	- 인증절차는 이름과 전화번호를 입력하여 일치하면 재수정 절차로 진행한다
- 리뷰관리.review
	- 리뷰 게시판에 게시글을 작성하였을 때 
		분류/ 제목/ 글쓴이/ 내용/ 날짜/ 조회수 순으로 작성한다  
			- 분류는 관리자가 생성해 놓은 것을 선택하여 기입한다
            - 제목은 한문자,영대소문자,숫자,툭수문자(!@#$%&) - 60자 이하로 한다 
            - 글쓴이는 게시글 작성자의 아이디를 기입한다 
            - 내용은 1000자 내로 기입 가능하며 사진과 동영상 링크 첨부가 가능하다
            - 날짜는 작성 날짜를 기준으로 한다 
            - 조회수는 회원, 비회원의 조회할 때마다 1씩 증가한다  
- 뉴스관리.news
	- 뉴스는 관리자만 생성/수정/삭제가 가능하다 
	- 뉴스 게시글을 작성했을 때
		제목/ 글쓴이/ 내용/ 날짜/ 조회수 순으로 작성한다 
        - 내용을 작성할 때는 출저 링크를 작성한다 
- 전문지식.opinion 
	- 전뮨지식 게시판에 게시글을 작성했을 때 
		분류/ 제목/ 글쓴이/ 내용/ 날짜/ 조회수 순으로 작성한다  
- 커뮤니티.community
	- 커뮤니티 게시판에 게시글을 ㄷ작성했을 때 
		분류/ 제목/ 글쓴이/ 내용/ 날짜/ 조회수 순으로 작성한다
	- 글쓴이의 최근 일주일 간의 이전 글의 제목을 출력한다 
- 숍.shop
	- 공동구매는 회원과 관라지만 조회 가능하다 
	- 공동구매 게시글을 작성했을 때
		분류/ 제목/ 글쓴이/ 내용/ 날짜/ 조회수 순으로 작성한다 
	- 내용은 구매 물품에 대한 내용을 입력하며 공동구매자 리스트를 작성한다 
    - 공동구매 리스트는 회원의 아이디와 이메일, 전화번호, 구매물품을 회원 개인에게 입력받는다 
    - 회원이 공동구매 리스트를 조회 시 아이디만 출력한다 
- 이벤트.event
	- 이벤트 게시글은 관리자만 작성 가능하며 작성할 때 
		분류/ 제목/ 글쓴이/ 내용/ 날짜/ 조회수 순으로 작성한다 
	- 이벤트 참여는 댓글로만 가능하며 비밀글만 작성 가능하다
    - 비밀글 조회는 관리자와 댓글 작성자만 가능하다 
- 댓글.comment / private comment
	- 댓글은 회원과 관리자만 작성 가능하며 일반댓글과 비밀댓글로 분류된다 
    - 댓글 중 일반 댓글은 권한 없이 조회가 가능하다 
    - 댓글 중 비밀 댓글은 관리자와 작성자만 조회 가능하다 
    - 댓글 작성 시 아이디/ 직성일자/ 내용 이 출력된다
    - 댓글은 300자 이하로 작성 가능하다 
- 사이트의 유튜브로 이동.youtube
	- 유튜브 선택을 실행하면 유튜브 채널 홈으로 이동 가능하다 
*/
